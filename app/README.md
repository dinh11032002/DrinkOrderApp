# Drink Order App

Ứng dụng Android hỗ trợ nhân viên phục vụ quản lý và theo dõi đơn hàng trong quán theo thời gian thực.

## Tính năng

- Quản lý bàn: xem trạng thái, phân quyền theo nhân viên
- Tạo và quản lý đơn hàng theo từng bàn
- Đồng bộ dữ liệu thời gian thực với Firestore
- Thông báo khi đơn hàng hoàn thành
- Lưu trữ offline với Room Database
- Hàng đợi xử lý đơn hàng tuần tự

## Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Architecture:** MVVM
- **Database:** Room (local), Firestore (remote)
- **Async:** Coroutines, Flow
- **Other:** Navigation Component, ViewModel, StateFlow
